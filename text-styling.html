Text Styling

Text styling in CSS involves changing the appearance of text on a webpage. Properties such as color, font-size, font-family, font-weight, text-decoration, and text-align can be used to enhance the readability and aesthetic of the content.

1. Changing Text Color

h1 {
    color: blue;
}

This sets the text color of all h1 elements to blue.

2. Changing Font Size

p {
    font-size: 16px;
}

This sets the font size of all p elements to 16 pixels.

3. Changing Font Family

body {
    font-family: Arial, sans-serif;
}

This sets the font family of the entire document to Arial or the default sans-serif font.

4. Changing Font Weight

h2 {
    font-weight: bold;
}

This makes the text of all h2 elements bold.

5. Text Decoration

a {
    text-decoration: underline;
}

This adds an underline to all a (anchor) elements.

6. Text Alignment

h3 {
    text-align: center;
}

This centers the text of all h3 elements.

7. Text Transformation

p {
    text-transform: uppercase;
}

This converts the text of all p elements to uppercase letters.

8. Line Height

p {
    line-height: 1.5;
}

This sets the line height of all p elements to 1.5 times the font size.

9. Letter Spacing

h1 {
    letter-spacing: 2px;
}

This adds 2 pixels of space between the letters of all h1 elements.

10. Word Spacing

p {
    word-spacing: 4px;
}

This adds 4 pixels of space between the words of all p elements.
