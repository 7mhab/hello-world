Colors & Backgrounds

Colors and backgrounds in CSS can be controlled using properties such as color, background-color, background-image, background-repeat, and background-size. These properties allow for the customization of the foreground and background elements to create visually appealing designs.

1. Setting Text Color

p {
    color: #ff0000;
}

This sets the text color of all p elements to red using a hex code.

2. Setting Background Color

body {
    background-color: #f0f0f0;
}

This sets the background color of the entire document to a light gray color.

3. Setting Background Image

div {
    background-image: url('background.jpg');
}

This sets a background image for all div elements.

4. Background Repeat

div {
    background-repeat: no-repeat;
}

This ensures that the background image does not repeat.

5. Background Size

div {
    background-size: cover;
}

This scales the background image to cover the entire container, while maintaining its aspect ratio.

6. Background Position

div {
    background-position: center;
}

This positions the background image at the center of the container.

7. Using RGBA for Background Colors

div {
    background-color: rgba(255, 0, 0, 0.5);
}

This sets the background color to red with 50% opacity using RGBA.

8. Using Gradients

div {
    background: linear-gradient(to right, red, yellow);
}

This creates a linear gradient background that transitions from red to yellow.
