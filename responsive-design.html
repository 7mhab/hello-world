Responsive Design

Responsive design in CSS ensures that webpages look good on all devices. Media queries are used to apply different styles based on the device's screen size, orientation, and resolution. This technique helps create a seamless user experience across various devices such as desktops, tablets, and smartphones.

1. Media Queries

Media queries allow you to apply CSS rules based on the characteristics of the device or screen, such as its width, height, or orientation. 

@media only screen and (max-width: 600px) {
    body {
        background-color: lightblue;
    }
}

This media query applies a light blue background color to the body element when the screen width is 600 pixels or less.

2. Fluid Grids

Fluid grids use relative units like percentages instead of absolute units like pixels to create flexible layouts. 

img {
    max-width: 100%;
    height: auto;
}

This ensures that images scale with the size of their containing element, maintaining their aspect ratio.

3. Flexible Layouts

Flexible layouts adapt to the screen size using properties like flexbox or grid layout. 

.container {
    display: flex;
    flex-wrap: wrap;
}

This creates a flexible container that can adjust its layout based on the screen size.

4. Responsive Typography

Responsive typography adjusts the font size based on the screen size. 

h1 {
    font-size: 2em;
}
@media (max-width: 600px) {
    h1 {
        font-size: 1.5em;
    }
}

This adjusts the font size of h1 elements to 1.5em when the screen width is 600 pixels or less.

5. Viewport Meta Tag

The viewport meta tag ensures that the website is properly scaled on mobile devices. 

<meta name="viewport" content="width=device-width, initial-scale=1.0">

This tag ensures that the viewport width matches the device's screen width and sets the initial zoom level to 1.0.

6. Responsive Images

Responsive images can adapt to different screen sizes using the srcset attribute. 

<img src="small.jpg" srcset="small.jpg 480w, medium.jpg 768w, large.jpg 1024w" alt="Responsive Image">

This example specifies different image sources for different screen widths.
