Variables & Data Types

Description: Variables in JavaScript store data that can be manipulated and referenced later. They can be declared using var, let, or const.

1.Variables

    Details:

  • var: Has function scope. Variables declared with var are hoisted to the top of their scope and can be re-declared and updated.
  • let: Has block scope. Introduced in ES6, let allows you to declare variables that are limited in scope to the block, statement, or expression where they are used.
  • const:Also has block scope. Variables declared with const cannot be re-assigned, which means the reference to the variable remains constant, though the value might still be mutable if it's an object.

2.Data Types

  • Primitive Types:
    • String:Represents textual data. e.g., "Hello, World!"
    • Number:Represents both integer and floating-point numbers. e.g., 42, 3.14
    • Boolean:Represents logical entities and can have two values: true or false.
    • Null:Represents the intentional absence of any object value.
    • Undefined:Indicates that a variable has not been assigned a value.
    • Symbol:Introduced in ES6, used to create unique identifiers for objects.
    • BigInt:Allows representation of whole numbers larger than 2^53 - 1.
  • Non-primitive Types:
    • Object:Collections of key-value pairs. Objects can include other objects, functions, and primitive values.
    • ArrayList-like objects used to store multiple values in a single variable.

3.Operators

    Details:

    • Arithmetic Operators:
      • + (Addition)
      • - (Subtraction)
      • * (Multiplication)
      • / (Division)
      • % (Modulus)
      • ** (Exponentiation, ES6)
  • Comparison Operators:
    • == (Equal to, loose equality)
    • === (Equal value and equal type, strict equality)
    • != (Not equal to, loose inequality)
    • !== (Not equal value and not equal type, strict inequality)
    • > (Greater Than)
    • < (Smaller Than)
    • >= (Greater than Or equal)
    • <= (Smaller than Or equal)
  • Logical Operators:
    • && (Logical AND)
    • || (Logical OR)
    • ! (Logical NOT)
  • Assignment Operators
    • = (Assignment)
    • += (Addition Assignment)
    • -= (Subtraction assignment)
    • *= (Multiplication assignment)
    • /= (Division assignment)
    • %= (Modulus assignment)
