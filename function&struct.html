Functions & Structures :

Description:Functions are blocks of code designed to perform specific tasks. They can be defined using the function keyword or arrow function syntax (=>).

Structure:Control structures manage the flow of the program: if, else if , else ,switch

JavaScript (Function Decleration)

function greet() {
console.log('Hello, World!');
}
JavaScript (Function Expressions)

const greet = function() {
console.log('Hello, World!');
};
JavaScript (Arrow Function)

const greet = () => {
console.log('Hello, World!');
};

Structure :

Structure:Control structures manage the flow of the program: if, else if , else ,switch
